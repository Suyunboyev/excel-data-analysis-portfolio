# WSJ Data Analysis (Excel)

## Project Overview
This project focuses on analyzing Wall Street Journal (WSJ) data using Microsoft Excel to uncover trends, patterns, and key insights through exploratory data analysis.

## Dataset
- Source: WSJ-related public dataset
- Type: Tabular data
- Focus: Trend analysis and summary statistics

## Objectives
- Clean and organize raw data
- Perform exploratory data analysis (EDA)
- Identify meaningful trends and patterns
- Visualize insights using charts

## Excel Techniques Used
- Data cleaning and formatting
- Pivot Tables
- Formulas (IF, COUNTIF, AVERAGEIF)
- Charts and visual analysis

## Key Insights
- Clear trends and patterns identified through aggregated views
- Excel pivot analysis highlights key changes over time
- Visualizations simplify complex data interpretation
