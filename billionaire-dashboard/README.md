# Billionaire Dashboard (Excel)

## Project Overview
This project presents an interactive Excel dashboard analyzing global billionaire data. The dashboard provides insights into wealth distribution, industries, countries, and net worth trends.

## Dataset
- Source: Public billionaire dataset
- Records: Global billionaires
- Key features: Name, Country, Industry, Net Worth, Age

## Objectives
- Analyze the distribution of billionaires by country and industry
- Identify top industries by total net worth
- Visualize key KPIs in a clear and interactive dashboard

## Excel Techniques Used
- Pivot Tables
- Advanced formulas (IF, VLOOKUP / XLOOKUP)
- Conditional Formatting
- Charts and KPI visualization
- Dashboard design

## Key Insights
- A small number of countries account for the majority of global billionaire wealth
- Technology and finance dominate in total net worth
- Significant wealth concentration is visible across industries
