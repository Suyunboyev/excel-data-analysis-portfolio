# Sales Performance Dashboard (Excel Analysis)

## Project Overview
This project focuses on transforming raw sales data into an interactive and actionable **Sales Performance Dashboard**. The goal was to analyze key business metrics, identify sales trends, and provide data-driven insights to help stakeholders make informed decisions.

## Key Features & Tools
* **Data Cleaning & ETL:** Performed using **Power Query** to handle missing values, format dates, and remove duplicates.
* **Data Analysis:** Utilized **Pivot Tables** and **Pivot Charts** for dynamic data summarization.
* **Advanced Formulas:** Implemented complex logic using `XLOOKUP`, `SUMIFS`, and `IFERROR` functions.
* **Interactivity:** Integrated **Slicers** and **Timelines** to allow users to filter data by region, category, and time periods instantly.

## Key Performance Indicators (KPIs) Analyzed
1.  **Total Revenue & Profitability:** Tracking year-over-year (YoY) growth.
2.  **Top Performing Products:** Identifying best-sellers and high-margin items.
3.  **Regional Sales Distribution:** Mapping sales across different geographical locations.
4.  **Sales Representative Performance:** Ranking staff based on targets achieved.
5.  **Monthly Trend Analysis:** Visualizing seasonal peaks and sales dips.

## Project Structure
* `Sales_Performance_Analysis.xlsx` - The main interactive Excel file.
* `Screenshots/` - High-resolution images of the dashboard.
* `Documentation/` - A brief summary of findings and business recommendations.
